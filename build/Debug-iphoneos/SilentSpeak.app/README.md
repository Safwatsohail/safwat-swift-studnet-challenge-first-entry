# ASL Images Folder

Place your ASL images here with the following naming convention:

## Letters
- ASL_A.jpg
- ASL_B.jpg
- ASL_C.jpg
- ... (all letters A-Z)

## Numbers  
- ASL_0.jpg
- ASL_1.jpg
- ASL_2.jpg
- ... (numbers 0-10)

## Words
- ASL_HELLO.jpg
- ASL_ANGRY.jpg
- ASL_PEACE.jpg
- ASL_GOODBYE.jpg
- ... (any words you have)

## Important Notes
- Use EXACTLY this naming format: ASL_[WORD].jpg
- All letters should be UPPERCASE
- Use underscores for multi-word phrases: ASL_THANK_YOU.jpg
- Supported formats: .jpg, .jpeg, .png

The app will automatically load these images when displaying ASL signs.
